# Nouns

Russian nouns are fully inflected for **number** and **case**, and have a fixed **lexical gender**.

### Inflection Tables

Example noun: **дом** (“house”, masculine)

|          | nominative | genitive | dative | accusative | instrumental | prepositional |
| -------- | ---------- | -------- | ------ | ---------- | ------------ | ------------- |
| singular | дом        | дома     | дому   | дом        | домом        | доме          |
| plural   | дома       | домов    | домам  | дома       | домами       | домах         |

Notes:

* Accusative forms vary by animacy; this table uses an **inanimate noun**.
* Gender is not displayed in the table because it is **lexically fixed**, not contrastive per row/column.

Adjectives inflect for **gender**, **number**, and **case**, and additionally for **degree**.

# Pronouns

Personal pronouns inflect for **person**, **number**, and **case**; some forms additionally encode **gender**.

## Inflection Tables

Example pronoun: **я** (“I”)

|                       | nominative | genitive | dative | accusative | instrumental | prepositional |
| --------------------- | ---------- | -------- | ------ | ---------- | ------------ | ------------- |
| first-person singular | я          | меня     | мне    | меня       | мной         | мне           |

Notes:

# Verbs

Russian verbs inflect for **person**, **number**, **tense**, **mood**, and **aspect**.
**Gender** is encoded only in **past tense** forms.

## Listed forms (synthetic)

Two verbs are used to demonstrate aspect contrast:

* **делать** (imperfective, “to do”)
* **сделать** (perfective, “to do / complete”)

Only synthetic forms are included.

## Inflection Tables

### Present tense (imperfective only)

Example verb: **делать**

|            | singular | plural  |
| ---------- | -------- | ------- |
| 1st person | делаю    | делаем  |
| 2nd person | делаешь  | делаете |
| 3rd person | делает   | делают  |
